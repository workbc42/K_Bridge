# 자주 묻는 질문

### Q1. 왜 기존 배달앱을 직접 만들지 않고 주문 대행 방식을 택했나요?

**A.** 초기 시장 검증 단계에서는 자체 배달앱 구축보다 기존 인프라를 활용하는 것이 비용·시간·리스크 측면에서 효율적입니다. 본 사업은 수요 검증을 최우선으로 하며, 검증 이후 단계적으로 시스템 고도화를 추진할 계획입니다.

### Q2. 수동 주문 구조는 확장성이 떨어지지 않나요?

**A.** 수동 주문 방식은 MVP 단계에서 의도적으로 선택한 전략입니다. 실제 주문 데이터와 운영 패턴을 확보한 후, 자동화 모듈을 도입하여 확장성을 확보할 계획입니다.

### Q3. 기존 배달 플랫폼 정책 변경 시 사업이 중단될 위험은 없나요?

**A.** 초기에는 기존 플랫폼 활용으로 안정적 운영을 확보하되, 장기적으로는 제휴 확대 및 자체 주문 관리 기능 고도화를 통해 의존도를 단계적으로 완화할 계획입니다.

### Q4. 법적·약관상 문제 소지는 없나요?

**A.** 본 사업은 고객의 명시적 요청에 따라 주문을 대행하는 중개 서비스로, 현행 법령 및 약관 범위 내에서 운영 가능하도록 검토하고 있습니다. 향후 서비스 확장 시에는 법률 자문을 병행할 예정입니다.

### Q5. 경쟁 서비스와의 차별점은 무엇인가요?

**A.** 외국인 대상 주문 대행에 특화된 구조와 실제 운영 경험을 기반으로 한 프로세스가 차별점입니다. 단순 번역 서비스가 아닌, 주문 처리까지 포함하는 관리 시스템을 제공합니다.

### Q6. 초기 고객은 어떻게 확보할 계획인가요?

**A.** 기존 주문 대행 운영을 통해 확보한 외국인 고객 네트워크를 기반으로 초기 수요를 확보하며, 외국인 커뮤니티·숙소·관광 채널 중심의 타겟 마케팅을 병행할 계획입니다.

### Q7. 정부 지원금이 없으면 사업이 불가능한가요?

**A.** 본 사업은 소규모 운영으로도 시작 가능한 구조이며, 정부 지원금은 서비스 고도화 및 자동화를 가속화하기 위한 목적입니다.

### Q8. 수익성은 충분히 검증되었나요?

**A.** 기존 주문 대행 운영 과정에서 고객의 반복 이용과 지불 의사를 확인하였으며, 이를 기반으로 수익 모델을 설계하였습니다.

### Q9. 향후 확장 방향은 무엇인가요?

**A.** 다국어 확대, 자동 주문 처리, 구독형 서비스 도입을 통해 외국인 대상 음식 주문의 표준 플랫폼으로 확장할 계획입니다.

### Q10. 대표자의 강점은 무엇인가요?

**A.** 본 사업은 대표자가 직접 외국인 주문 대행을 운영하며 축적한 경험을 기반으로 기획된 재창업 프로젝트로, 현장 이해도와 실행력이 강점입니다.

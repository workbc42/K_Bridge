# 프론트 기본 화면/디자인 골격

## 목표
- MVP 단계에서 소개/온보딩용 랜딩 페이지를 구성
- 텍스트 위주 정보 전달 + 디자인 톤 설정

## 디자인 방향
- 컬러: 따뜻한 뉴트럴 + 포인트 오렌지/딥블루
- 타이포: 헤드라인용 Serif + 본문용 Sans
- 배경: 그라디언트 + 소프트 블러

## 주요 섹션
1. Hero
   - 서비스 슬로건, 핵심 가치, CTA 버튼
2. 핵심 기능
   - 다국어, 결제, 실시간 추적 강조
3. 사용 흐름
   - 3단계 주문 과정
4. MVP 포커스
   - 현재 제공 범위 안내
5. CTA
   - 베타 참여 유도

## 파일 위치
- `frontend/app/page.js`
- `frontend/app/layout.js`
- `frontend/app/globals.css`

## 적용 사항
- `next/font/google`로 폰트 적용
- 커스텀 CSS 변수로 컬러/폰트 토큰 정의
- 기본 애니메이션(페이드 인 + 스태거) 적용

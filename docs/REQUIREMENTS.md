# 요구사항 정의서

## 1. 사용자 스토리

### 외국인 사용자
- 한국어를 몰라도 영어로 음식을 주문하고 싶다
- (Opt) 외국 카드로 결제하고 싶다
- 주문 상태를 실시간으로 확인하고 싶다

### 관리자
- 사용자의 주문 요청을 확인하고 싶다
- 배달앱에서 대행 주문하고 싶다
- 주문 상태를 업데이트하고 싶다

## 2. 기능 요구사항

### 2.1 사용자 기능
- [ ] 회원가입/로그인
- [ ] 음식점 검색
- [ ] 메뉴 선택 및 장바구니
- [ ] 주문하기
- [ ] 결제하기
- [ ] 주문 내역 조회

### 2.2 관리자 기능
- [ ] 주문 요청 확인
- [ ] 주문 상태 업데이트
- [ ] 고객 문의 응답

## 3. 비기능 요구사항
- 응답 시간: 2초 이내
- 동시 접속자: 100명
- 가용성: 99.9%
